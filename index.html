Shift-tolerant Perceptual Similarity Metric
Abhijay Ghildyal, Feng Liu
European Conference on Computer Vision (ECCV), 2022
arXiv  /  code  /  bibtex

We investigate a broad range of neural network elements and develop a shift-tolerant perceptual similarity metric that is consistent with human perception and more resistant to imperceptible misalignment between a pair of images than existing metrics.
